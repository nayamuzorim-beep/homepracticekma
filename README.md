# Green Budgeting Dashboard (greenbudeting.html)

이 저장소에는 "2023 글로벌 기후 정책 및 탄소인지예산 통합 보고서" 대시보드 페이지가 포함되어 있습니다. 다음 방법으로 외부에서 해당 HTML 페이지를 열어볼 수 있습니다.

## 직접 열기 (권장)
- GitHub Blob(파일 보기):
  https://github.com/nayamuzorim-beep/homepracticekma/blob/3e5f0c65359fff03e93006a73362832cc17aa3fd/greenbudeting.html
  - 이 링크는 GitHub의 파일 보기 페이지로 이동합니다. 상단의 "Raw" 버튼을 클릭하면 브라우저에서 HTML이 바로 렌더링됩니다.

- Raw 파일(브라우저에서 바로 렌더링 가능):
  https://raw.githubusercontent.com/nayamuzorim-beep/homepracticekma/3e5f0c65359fff03e93006a73362832cc17aa3fd/greenbudeting.html
  - 이 URL을 브라우저 주소창에 붙여넣으면 HTML 페이지가 직접 열립니다.

## 로컬에서 열기
1. 저장소를 클론하거나 `greenbudeting.html` 파일을 다운로드합니다.
2. 파일을 더블클릭하여 브라우저로 열거나, 로컬 HTTP 서버를 실행하여 확인할 수 있습니다. 예:
   - Python 3: `python3 -m http.server 8000`
   - 그 후 브라우저에서 `http://localhost:8000/greenbudeting.html` 열기

## GitHub Pages로 호스팅(선택)
- 더 안정적으로 외부에 공개하려면 GitHub Pages를 활용하세요. 방법 중 하나:
  1. `greenbudeting.html` 파일을 `docs/` 폴더로 옮기거나 저장소 루트에 둡니다.
  2. GitHub 저장소 설정(Settings) > Pages에서 브랜치(예: `main`)와 폴더(`root` 또는 `docs`)를 선택합니다.
  3. 설정 후 제공되는 GitHub Pages URL에서 페이지가 호스팅됩니다.
